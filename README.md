# CS50 AI - Introduction to Artificial Intelligence with Pyhton

Este repositório contém os projetos desenvolvidos ao longo do curso  
**CS50’s Introduction to Artificial Intelligence with Python**  
O curso explora algoritmos e conceitos fundamentais de inteligência artificial — como busca, aprendizado de máquina, otimização, redes neurais e processamento de linguagem natural — aplicados com programas em Python.

## Estrutura do Repositório

O repositório está dividido em pastas, onde cada pasta corresponde aos projetos desenvolvidos em cada módulo do curso.
